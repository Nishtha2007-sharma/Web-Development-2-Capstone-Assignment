# 🛡️ CyberSense

A Cybersecurity Awareness & Threat Intelligence Dashboard built with React + Vite.

🔗 **Live Demo:** [gentle-cucurucho-ad752f.netlify.app](https://gentle-cucurucho-ad752f.netlify.app)

## Features
- 🔍 Breach Checker — Search known public data breaches
- ⚠️ Threat Feed — Live CVE vulnerabilities from NIST
- 🌐 IP Scanner — Scan IPs and URLs for threats
- 🧠 Security Quiz — Test your cybersecurity knowledge
- 🌙 Dark Mode Toggle

## Tech Stack
React, Vite, Tailwind CSS, React Router, Axios, Context API
